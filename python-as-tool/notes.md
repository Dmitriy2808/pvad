## Установка пакетов

* easy_install
* pip3

  ```
  $ sudo pip3 install package_name
  ```

  Разбор.

  Обновление:

  ```
  $ sudo pip3 install -U package_name
  ```

  Установка в папку пользователя

  ```
  $ sudo pip3 install --user package_name
  ```

  Особенности именования в Windows.

* PyPI
* виртуальное окружение, virtualenv
